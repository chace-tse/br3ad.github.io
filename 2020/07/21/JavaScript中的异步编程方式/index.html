什么是异步（Asynchronous）？

简单来说就是一个任务分成两段，先执行一段，然后转而执行其他任务，等做好了准备，再回过头执行第二段。

比如，有一个任务是读取文件进行处理，异步的执行过程就是不连续的执行，就叫做异步（Asynchronous），相应地，连续的执行，就叫做同步（Synchronous）

JavaScript中异步编程的方法有：

  • 回调函数
  • 事件监听
  • 发布/订阅
  • promise 对象
  • generator(ES6)
  • async / await（ES7）
